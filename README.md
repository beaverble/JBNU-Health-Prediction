# JBNU 웨어러블 기반 건강 지표 예측 시스템

웨어러블 기기에서 수집되는 생체 데이터(혈압, 스트레스, 산소포화도)를 실시간으로 분석하고, 시계열 딥러닝 모델(LTSF-NLinear)을 활용하여 미래 건강 지표를 예측하는 통합 관리 시스템입니다.


## 1. 주요 기능

* **실시간 생체 데이터 수집**: 외부 API를 통해 웨어러블 사용자의 혈압(수축기/이완기), 스트레스 지수, 산소포화도(SpO2) 데이터를 자동으로 수집합니다.
* **데이터 정규화 및 전처리**: 수집된 다양한 범위의 생체 데이터를 분석에 적합하도록 Min-Max Scaling 기법을 사용하여 정규화합니다.
* **시계열 기반 지표 예측**: LTSF-NLinear 모델을 활용하여 과거 데이터를 바탕으로 향후 12시간 동안의 건강 지표 변화를 예측합니다.
* **예측 점수 산출 및 전송**: 예측된 혈압 수치 등을 바탕으로 건강 점수를 계산하고, 이를 다시 시스템 API를 통해 결과 서버로 전송합니다.
* **자동 스케줄링**: 매시간 정해진 시점에 예측 작업을 수행하며, 오류 발생 시 자동으로 재시작하는 안정적인 운영 환경을 제공합니다.

## 2. 프로젝트 구조

| 파일명 | 설명 |
| :--- | :--- |
| `main.py` | 시스템 엔트리 포인트 및 스케줄러 (매시간 예측 작업 실행 및 오류 관리) |
| `predict_health.py` | 학습된 LTSF-NLinear 모델을 로드하여 건강 지표 예측을 수행하는 핵심 로직 |
| `request_moduel.py` | 외부 API 연동 모듈 (사용자 조회, 생체 데이터 수집, 결과 전송 기능 담당) |
| `scaling_data.py` | 데이터 정규화(Scaling) 및 모델 출력값의 역변환(Inverse Scaling) 처리 |
| `Train_LTSFNLinear.ipynb` | 시계열 예측 모델(Linear, DLinear, NLinear)의 학습 및 성능 평가용 노트북 |

#### **2-1. 예측 파이프라인 (Daily)**
매시간 정각에 실시간 데이터를 바탕으로 예측을 수행합니다.
* **스케줄러 가동**: `main.py` 실행 시 정해진 주기마다 예측 작업이 시작됩니다.
* **사용자 및 데이터 수집**: `request_moduel.py`를 통해 현재 활성화된 웨어러블 사용자와 최근 생체 데이터를 수집합니다.
* **데이터 정규화**: `scaling_data.py`를 사용하여 수집된 원본 데이터를 모델 입력 형식에 맞춰 변환합니다.
* **건강 지표 예측**: `predict_health.py`에서 학습된 모델을 사용하여 향후 12시간의 수치를 예측합니다.
* **점수화 및 결과 전송**: 예측된 수치를 역변환하여 실제 값으로 복구한 뒤, 점수를 산출하여 시스템에 등록합니다.

## 3. 설치 및 요구 사항

#### 3-1.  **필수 라이브러리 설치**:

```
  pip install -r requirements.txt
```

#### 3-2.  **시스템 환경**:

* 본 시스템은 PyTorch 프레임워크를 기반으로 구축되었습니다.
* 모델 학습 및 대규모 예측 시 성능 향상을 위해 GPU 환경을 권장합니다.

## 4. 실행 방법

시스템의 자동 스케줄러를 시작하려면 다음 명령어를 실행하십시오.

```
python main.py
```

## 시스템 설정 가이드

* **인증 토큰**: API 호출 시 `X-Auth-token` 헤더가 사용됩니다. 각 스크립트 내의 토큰 정보가 유효한지 확인하십시오.
* **데이터 필터링**: 검침 데이터 중 결측치(NaN)가 많거나 데이터 개수가 부족한 가구는 예측 신뢰도를 위해 대상에서 자동으로 제외됩니다.
* **로그 확인**: 모든 작업 결과(성공/실패 메시지)는 시스템 로그 API를 통해 전송되어 모니터링됩니다.
